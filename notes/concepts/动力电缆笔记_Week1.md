## 一、大背景：为什么需要"动力"电缆？

### 1.1 固定式 vs 浮式风机的核心区别

```
固定式风机（Fixed Offshore Wind）          浮式风机（FOWT）
──────────────────────────────────────────────────────────
塔架固定在海床（水深 < 50-60m）            平台漂浮在海面（水深 60-300m+）
电缆静态埋在海底                           电缆必须跟随平台运动
无反复弯曲疲劳问题                         ↕ 随波浪6个方向运动（6DOF）
                                            动力电缆承受反复弯曲、扭转、张力
                                                    ↓
                                            疲劳 = 动力电缆主要失效模式
```

**为什么要去深水？**
- 浅水区（<50m）好的风力资源位置正在减少
- 深水区风力资源更稳定、更强
- 欧洲、日本近海 → 大部分适合建风场的地方水深超过60m

### 1.2 行业规模（来自白皮书 2024）
- 2030年目标：全球海上风电装机量达 **305 GW**
- 2033年目标：达 **480 GW**
- 动力电缆是制约浮式风电商业化的关键瓶颈之一

## 二、风场电缆系统：两种电缆

```
海上风电场电力传输路径：
┌─────────┐     阵列电缆      ┌──────────┐    出口电缆    ┌──────┐
│ 风机 1  │ ←───────────────→ │ 海上变电站│ ←──────────→ │  陆地  │
│ 风机 2  │     Inter-array   │（OSS）   │  Export cable  │  电网  │
│ 风机 3  │      cable        └──────────┘               └──────┘
└─────────┘
```

### 2.1 阵列间电缆（Inter-array Cable）
- **作用**：连接各风机 → 变电站
- **电压**：现在 36kV / 66kV → 未来升级到 **132kV**（支持更大风机）
- **形式**：交流（AC，三相）
- **设计**：浮式风机需要"动力"版本（能承受反复弯曲）

### 2.2 出口电缆（Export Cable）
- **作用**：变电站 → 陆地电网
- **电压**：220kV（交流 HVAC）或更高（直流 HVDC）
- **现状**：高压动力出口电缆是目前行业最大的技术空白！
- **挑战**：距离越远越需要直流（DC），但动力直流电缆技术尚未成熟

## 三、动力电缆的三种构型（★重点）

### 3.1 悬链线构型（Catenary Configuration）

```
平台
  ↘
    ↘  自然悬垂（像锁链一样垂下来）
      ↘
        ↘
          ──→ TDP（触底点）
```
- **优点**：结构简单，无附件
- **缺点**：TDP处张力大，疲劳寿命短；适合深水、平静海况

### 3.2 懒波构型（Lazy-Wave Configuration）★ 当前主流

```
平台
  ↘（下降段）
    ↙（Sag Bend 下弯点）
  ↑（浮力段：装有浮力体 Buoyancy Module）
  ↑
    ↘（Hog Bend 上弯点）
      ↘（触底段）
        ──→ TDP（触底点）
```
- **关键创新**：中段装上浮力体，使电缆形成"懒波"形状
- **优点**：浮力段像缓冲器，隔离平台运动→TDP疲劳寿命提升3-5倍
- **现状**：实际项目主流选择

### 3.3 W形构型（W-shape Configuration）
- 用浮力体 + 压载体把电缆悬在水中，完全不触底
- 适用特殊工况，不常见

## 四、动力电缆承受的载荷

### 4.1 三种基本载荷
```
1. 张力（Tension）          电缆被拉伸
2. 弯曲（Bending）          电缆被弯折 → 最主要的疲劳来源
3. 扭转（Twisting）         电缆被扭转 → 铠装钢丝扭转耦合
```

### 4.2 Near方向 vs Far方向（★设计关键）
```
Near（近端）：平台向触底点方向移动
  → 电缆被压缩 → 可能发生压屈（Buckling）

Far（远端）：平台远离触底点方向移动  
  → 电缆被拉伸 → 张力增大
```
两种方向都要满足设计要求！

### 4.3 其他载荷来源
- 海流（Current）：横向力 → 涡激振动（VIV）
- 海洋生物附着（Marine Growth）：增加电缆重量 → 改变构型形态
- 平台6个方向的运动（Surge/Sway/Heave/Roll/Pitch/Yaw）

## 五、疲劳热点：4个危险位置

```
危险程度：挂接点 > TDP > Sag Bend ≈ Hog Bend

1. 挂接点（Hang-off Point）★★★ 最严重
   ├─ 位置：电缆连接到平台的地方
   ├─ 原因：平台所有运动直接传入，弯曲最剧烈
   └─ 保护：弯曲加强件（Bend Stiffener）

2. 下弯点（Sag Bend）
   ├─ 位置：浮力段下边缘
   └─ 原因：浮力体边缘刚度突变 → 应力集中

3. 上弯点（Hog Bend）
   ├─ 位置：浮力段上边缘  
   └─ 原因：同Sag Bend，刚度突变

4. 触底点（TDP，Touchdown Point）★★ 你的研究切入口
   ├─ 位置：电缆接触海床的位置
   ├─ 原因：海床土摩擦 + 平台Surge运动→TDP反复移动
   └─ 特殊：地质条件（土类型/地形）在此处起关键作用！
```

## 六、电缆内部结构

```
动力电缆截面（从外到内）：
┌──────────────────────────────────────┐
│  外护套（Outer Sheath）               │ 聚乙烯，防腐防水
│  铠装层×2（Double Armour）           │ 钢丝螺旋缠绕（动力电缆需双层！静态只需单层）
│  绝缘层（Insulation/XLPE）            │ 交联聚乙烯，电气绝缘
│  导体（Conductor）                    │ 铜/铝，传输电力
│  光纤（Fibre Optic，可选）            │ 状态监测 + 通信
└──────────────────────────────────────┘

关键力学参数：
EI  = 弯曲刚度（kNm²）  越大越难弯，决定电缆形态
EA  = 轴向刚度（MN）    决定拉伸变形量
GJ  = 扭转刚度          铠装钢丝扭转-弯曲耦合
MBR = 最小弯曲半径（m） 超过 → 绝缘层损伤，是设计底线
```

**注意**：动力电缆需要**双层铠装**，而固定式静态电缆只需单层，这直接导致成本和重量增加。

## 七、电气绝缘老化问题（Ageing）

两种主要失效模式：
1. **水树（Water Treeing）**：水分渗入绝缘层 + 交流电场 → 绝缘变弱
2. **电树（Electrical Treeing）**：气体通道导致放电 → 最终绝缘失效（短路）

预防方法：
- 干式设计（Dry Design）：铅制防水层，效果好但重
- 湿式设计（Wet Design）：无铅，更轻便宜，但需要其他防水材料

### 11.1 阵列间电缆的三种连接方式（笔记原来完全缺失）

风机之间的阵列电缆有三种接法：

```
方案①：全动力电缆（Full Dynamic）
[FOWT-A] ══════════════════════ [FOWT-B]
← 整段全部用动力电缆，成本最高 →

方案②：动力+静态+现场接头/连接器
[FOWT-A] ══╤──────────────╤══ [FOWT-B]
          接头          接头
← 两端动力，中间静态，现场海上连接 →

方案③：动力+静态+工厂接头（整段预制）
[FOWT-A] ══╪──────────────╪══ [FOWT-B]
          工厂接头      工厂接头
← 在陆上工厂预制好整段，作为单段安装 →
```

**选择依据**：动力电缆比静态贵2-3倍，中间大段如果在海底静止就没必要用动力电缆。
**工厂接头 vs 现场接头**：工厂接头质量更稳定（受控环境制作），现场接头受海上天气影响。

### 11.2 风场全局布局：三种类型（笔记原来完全缺失）

```
菊花链（Daisy Chain）— 当前主流，所有实际风场都用这种
[F1]─[F2]─[F3]─[F4]─[变电站]
优点：技术成熟  缺点：一台故障影响全链（单点故障）

鱼骨型（Fishbone）
[F1]─┐
[F2]─┤─[主干]─[变电站]
[F3]─┘
优点：冗余性↑，电缆总量↓  缺点：需要水下连接器（未成熟）

星型（Star）
[F1]─┐
[F2]─┤─[海底集线器HUB]─[变电站]
[F3]─┘
优点：每台独立，冗余最好  缺点：需要海底集线器（技术最不成熟）
```

**菊花链的运维风险**：一台FOWT拖回港口大修时，相邻风机的电缆要从海底拆下临时存放在海床或浮标上，整个阵列可能停发电。这叫"拖回港口维修策略（Tow-to-port repair strategy）"的连锁影响。

### 11.3 固定式风电的CPS问题（笔记原来完全缺失）

**CPS = Cable Protection System（电缆保护系统）**

即使固定式风电的电缆埋在海底，在塔架出口处有一小段暴露在水中，受海流/波浪/冲刷（Scour）作用，需要CPS保护：

| 组件 | 作用 |
|---|---|
| Bend Stiffener（弯曲加强件）| 锥形套，防止出口处过度弯曲 |
| Bend Restrictor（弯曲限制件）| 连锁套，限制最大弯曲角度 |
| I-tube | 导管，引导电缆从结构内穿出 |
| J-tube | J形弯管，从海床斜向进入结构 |

**重要数字**：近年CPS问题已影响约**10座**固定式风场，直接推动了保险市场收紧。这是浮式风电要吸取的教训。

### 11.4 整体工程方法（Holistic Engineering Approach）

笔记第十节提到"数字孪生"，但没有提核心设计理念：

```
❌ 错误做法（分开设计）：
  风机团队 + 浮台团队 + 系泊团队 + 电缆团队 各自为政
  → 每个子系统各自达标，但系统整合出问题
  （例：浮台运动满足系泊要求，但超出电缆承受范围）

✅ 正确做法（整体工程）：
  风机 + 浮台 + 系泊 + 电缆 → 作为耦合系统一起优化
  → 某个部分的改变会影响其他部分 → 需要迭代设计
```

这是白皮书的**核心理念**，贯穿全文。

### 11.5 Carbon Trust关键调查结果

**调查1（规范缺口量化）**：
审查了62份相关技术文献，只有**6份**（ORE Catapult统计8份）直接适用于动力海底电力电缆。
→ 约10%的现有知识体系能用于浮式风电动力电缆设计。

**调查2（标准覆盖范围）**：
- 现有标准只覆盖干式静态 132kV ✓
- 湿式静态 132kV → 需要显著改进
- 动态 132kV → 需要额外测试（基本空白）

**调查3（大直径的反直觉结论）**：
电缆直径越大 → 水动力载荷越大 → 内部应力幅越大 → **疲劳寿命反而更短！**
→ 高电压（需大截面导体）和高疲劳寿命之间存在矛盾。

### 11.6 HVAC vs HVDC 选择依据（笔记原来只列举，没有解释）

| | HVAC（交流高压）| HVDC（直流高压）|
|---|---|---|
| 适用距离 | < 100 km | > 100 km |
| 优点 | 技术成熟，变压器便宜 | 长距离损耗小 |
| 缺点 | 距离长时电容电流大，损耗不可接受 | 换流站贵（数亿欧元）|
| 典型应用 | 近海（欧洲多数现有风场）| 远海（如苏格兰外海）|

日本近海示范项目（福岛附近，水深100-200m）→ 距离通常<100km → HVAC足够。

### 12.2 莫里森方程（Morison Equation）——两方向共用

电缆和立管计算流体力都用莫里森方程（因为两者直径远小于波长，属于细长体）：

```
单位长度流体力：
F = F_惯性 + F_拖曳

F = ρ·Cm·A·v̇  +  ½·ρ·Cd·D·|v_rel|·v_rel

参数：
Cm ≈ 2.0（附加质量系数）
Cd = 0.6~1.2（拖曳力系数，依Reynolds数）
A  = 截面积（m²）
D  = 外径（m）
v_rel = 流体速度 - 结构速度（相对速度）
```

- **B-4电缆**：用于计算阵列间和触底段所受流体力 → 影响TDP处载荷
- **C-3立管**：用于计算沿立管全长的流体力（矢量形式，见立管笔记 §四.3）

> 📎 详细推导和矢量形式：见 `深水立管/week1/精读_懒波立管综述_Section1_引言.md` §2.2.3

### 12.3 疲劳分析流程——两方向完全共通

```
Step 1：OrcaFlex时域仿真
  → 输出各位置弯矩时间序列 M(t)

Step 2：Rainflow计数
  → 统计弯矩幅值的分布 Δσ_i, n_i

Step 3：S-N曲线查阅
  → 得到各幅值对应的疲劳寿命 N_i（次）

Step 4：Miner则线性叠加
  → D = Σ(n_i/N_i)
  → D ≥ 1 → 失效

设计要求：D < 1/DFF（设计疲劳系数，DNV规范中通常=10）
```

**B-4 vs C-3的差异只有一点**：
- B-4：D_total = D_wave（只有波浪疲劳）
- C-3：D_total = D_wave + D_VIV（还要加SHEAR7计算的VIV疲劳！）

> 📎 详细对比：见 `深水立管/week1/笔记_Week1.md` §五
